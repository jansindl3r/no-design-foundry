## Frontend

1. Setup

    1. Create a `.env` file by running:
        ```sh
        cp .env.example .env
    1. Install the necessary dependencies:
        ```sh
        npm install
        ```
1. Development
    1. Start the development server:
        ```sh
        npm run dev
        ```
    1. Open your browser and navigate to [localhost:3000](localhost:3000).
1. Enjoy developing!
